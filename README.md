# Yu-Gi-Oh! Master Duel 中文卡查

## Download

你可以在[Releases](https://github.com/SkywalkerJi/mdt/releases/latest)下载预打包的版本。

## Usage
```
打开游戏后，运行mdt.exe，根据提示使用快捷键。
卡组界面和决斗界面需要 ctrl+s 切换模式。
```
